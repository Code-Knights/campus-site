# Pendahuluan
 - Salah satu kriteria penting dari sebuah antarmuka adalah
tampilan `menarik`

- harus memiliki oleh perancang tampilan
  - Jiwa seni yang memadai
  - Mengetahui selera user secara umum

- Perancang tampilan _`harus`_ **mendokumentasikan**
semua pekerjaan yang yang ia kerjakan selama ini, Sebagai bahan evaluasi pembuatan tampilan yang baru

> PROGRAM YANG BUAT BUKAN UNTUK ANDA, NAMUN UNTUK ORANG LAIN


## Spesial Purpose Software
 Yaitu program aplikasi untuk keperluan khusus
dengan user

## General Purpose Software
yaitu program aplikasi yg akan digunakan oleh berbagai macam user

- perancang dapat memaksa kepada user untuk menerima tampilan interface, walaupun bisa dilaku
- Salah satu bisa melakukan `kustomisasi'
